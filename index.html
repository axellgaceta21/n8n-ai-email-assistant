PROJECT OVERVIEW

Building Smarter Customer Support Using AI

⚠️

Problem

Business owners spend hours manually reading, categorizing and replying to emails. This repetitive process slows response times and reduces productivity.

🤖

Solution

An AI-powered email assistant that reads emails, extracts company knowledge from the website, classifies customer intent, and drafts professional responses automatically.

STEP 1

Gmail Trigger

Automatically listens for new incoming emails.

  • Extract Subject
  • Extract Sender Email
  • Extract Email Body
  • Real-time Monitoring
{
"subject":"Internet Installation Inquiry",

"sender":"customer@email.com",

"body":"When can my internet be installed?"
}
STEP 2

HTTP Request

Fetches the company website to provide the AI with up-to-date business information.

  • GET Request
  • Returns Website HTML
  • Live Knowledge Source
  • No Hardcoded Data
STEP 3

HTML Extract

Extracts readable information from the website.

  • Pure Fiber Internet
  • Unlimited Internet Plans
  • WiFi Hotspots
  • Free Installation
  • Technical Support
  • PisoWiFi Solutions
{
"website_text":

"Pure Fiber Internet,

Unlimited Internet Plans,

WiFi Hotspots,

Free Installation..."
}
STEP 4

Edit Fields

Combines email data and website information into a single payload for the AI Agent.

  • Email Subject
  • Sender
  • Email Body
  • Website Content
  • Structured Payload
{
"subject":"...",

"sender":"...",

"body":"...",

"website":"Company services..."
}
STEP 5

AI Agent (OpenAI)

Acts as an AI customer support representative. Uses website knowledge and customer email context to generate intelligent responses.

  • Summarize Email
  • Categorize Intent
  • Use Website Knowledge
  • Generate Reply
  • Return JSON Output
{
"summary":

"Customer is asking about installation schedule.",

"category":

"Installation Scheduling",

"reply":

"Thank you for contacting us..."
}
STEP 6

Response Formatting

Formats the AI response before sending it to Gmail.

  • Preserve line breaks
  • HTML-safe text
  • Email Template
  • Responsive Formatting
STEP 7

Gmail Draft Message

Drafts a professional branded email ready for human review.

  • Human Review
  • Company Logo
  • Responsive Design
  • Service Icons
  • Professional Signature
